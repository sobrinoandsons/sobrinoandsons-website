# Sobrino and Son's

## INTRO

The purpose of this project is to provide the auto mechanic company Sobrino and Son's LLC with a simple website.

## Description

The website will be a relatively simple, static, multi-page project. We will make use of html, css, js, GitHub, VS Code, and Cloudflare.

The language of the site will be Spanish. It will have a single page. At the top there will be a navbar with the logo and name of the business on the left. Then, on the right it will have the links to the sections Inicio, Servicios, Reseñas, and Contáctenos. After the navbar, we will have a hero with an image that contains the logo and name of the business. Bellow that image, we will have a brief description of the business. After that, we will have the sections mentioned above. Then we will have a footer.

The color palette of the website will be white as background, red as color 1 and black as color 2. Other accents of these colors may be used.
